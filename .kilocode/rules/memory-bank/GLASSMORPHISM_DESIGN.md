# Glassmorphism Design Implementation

## Обзор

Реализован современный темный интерфейс с эффектом стекла (glassmorphism) на основе эталонного изображения. Дизайн включает полупрозрачные элементы с эффектом матового стекла, градиентные фоны и золотые акценты.

## Файлы проекта

### Основные файлы
- **`static/glass-theme.css`** - новый файл стилей с glassmorphism компонентами
- **`static/styles-old.css`** - резервная копия старых стилей
- **`static/styles.css`** - оригинальные стили (сохранены для совместимости)
- **`includes/layout.php`** - обновлен для подключения glass-theme.css и нижней навигации
- **`index.php`** - модифицирован с применением стеклянных компонентов

## Ключевые компоненты

### 1. Градиентный фон

Основной фон страницы с многослойным градиентом:

```css
background: linear-gradient(
  135deg,
  #0a0e27 0%,
  #1a1f3a 25%,
  #2563eb 50%,
  #1e2742 75%,
  #0a0e27 100%
);
background-attachment: fixed;
```

Дополнительные радиальные градиенты через `::before`:
- Синее свечение в верхнем левом углу (20% 30%)
- Золотое свечение в правом нижнем углу (80% 70%)

### 2. Стеклянные карточки

#### `.glass-card` - базовая стеклянная карточка
```css
background: rgba(255, 255, 255, 0.1);
backdrop-filter: blur(10px);
border: 1px solid rgba(255, 255, 255, 0.2);
border-radius: 20px;
box-shadow: 
  0 8px 32px rgba(0, 0, 0, 0.37),
  inset 0 0 0 1px rgba(255, 255, 255, 0.1);
```

**Hover эффект:**
- Увеличение прозрачности до 0.15
- Поднятие на 5px (`translateY(-5px)`)
- Усиление тени

#### `.glass-card-glow` - карточка с внутренним свечением
```css
background: rgba(30, 39, 66, 0.6);
backdrop-filter: blur(15px);
border: 1px solid rgba(255, 255, 255, 0.18);
border-radius: 24px;
```

Включает внутренние тени для эффекта глубины.

#### `.glass-program-card` - карточки программ/проектов
Специализированные карточки с:
- Верхней золотой полосой (появляется при hover)
- Анимацией поднятия на 8px
- Усиленными тенями при наведении

#### `.glass-mission-card` - карточки миссии/о фонде
```css
background: linear-gradient(
  135deg,
  rgba(212, 175, 55, 0.15) 0%,
  rgba(74, 144, 226, 0.1) 100%
);
```

Градиентный фон от золотого к синему.

### 3. Нижняя навигация

#### `.glass-bottom-nav`
Фиксированная нижняя панель навигации:

**Позиционирование:**
```css
position: fixed;
bottom: 2rem;
left: 50%;
transform: translateX(-50%);
z-index: 1000;
```

**Glassmorphism эффект:**
```css
background: rgba(30, 39, 66, 0.7);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.18);
border-radius: 30px;
```

**Тени:**
- Внешняя тень для глубины
- Внутреннее свечение сверху
- Синее свечение вокруг (`0 0 20px rgba(74, 144, 226, 0.3)`)

#### `.glass-nav-item` - элементы навигации
```css
padding: 0.75rem 1.5rem;
color: var(--glass-text-secondary);
border-radius: 20px;
```

**Состояния:**
- **Hover:** фон `rgba(255, 255, 255, 0.1)`, поднятие на 2px
- **Active:** фон с золотым оттенком, золотой цвет текста, свечение

#### `.glass-nav-cta` - золотая кнопка действия
```css
background: linear-gradient(135deg, #fde9a9;/*   #d4af37; */ 0%, #ffb340 100%);
color: #0a0e27;
box-shadow: 0 4px 15px rgba(212, 175, 55, 0.4);
```

При hover:
- Обратный градиент
- Поднятие на 3px
- Усиление свечения

### 4. Иконки

## Анимации

### 1. Появление снизу вверх
```css
@keyframes glassSlideUp {
  from {
    opacity: 0;
    transform: translateY(30px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

**Классы:**
- `.glass-animate` - базовая анимация
- `.glass-animate-delay-1` - задержка 0.1s
- `.glass-animate-delay-2` - задержка 0.2s
- `.glass-animate-delay-3` - задержка 0.3s

### 2. Пульсация золотого свечения
```css
@keyframes goldPulse {
  0%, 100% {
    box-shadow: 0 4px 15px rgba(212, 175, 55, 0.3);
  }
  50% {
    box-shadow: 0 4px 25px rgba(212, 175, 55, 0.6);
  }
}
```

## Совместимость браузеров

### Поддержка backdrop-filter

**Поддерживается:**
- Chrome 76+
- Edge 79+
- Safari 9+ (с префиксом -webkit-)
- Firefox 103+

**Префиксы:**
```css
backdrop-filter: blur(10px);
-webkit-backdrop-filter: blur(10px);
```

### Фоллбэк для старых браузеров

Если `backdrop-filter` не поддерживается, карточки будут иметь более непрозрачный фон благодаря базовому `background` свойству.

## Производительность

### Оптимизации

1. **Фиксированный фон:**
   ```css
   background-attachment: fixed;
   ```
   Предотвращает перерисовку градиента при скролле.

2. **will-change для анимаций:**
   Применяется автоматически браузером для transform свойств.

3. **CSS переменные:**
   Использование CSS custom properties для быстрого изменения темы.

4. **Hardware acceleration:**
   Все трансформации используют `transform` вместо `top/left` для GPU ускорения.
