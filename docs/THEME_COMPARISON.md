## 🎯 Психология цветов

### Тёмная тема 🌙
- **Espresso фон** → Премиальность, серьёзность, элегантность
- **Светлые карточки** → Фокус на контенте, читаемость
- **Золотые акценты** → Ценность, важность, тепло

### Светлая тема ☀️
- **Light-yellow фон** → Оптимизм, тепло, приветливость
- **Золотые карточки** → Энергия, позитив, ценность
- **Медно-красные карточки** → Страсть, действие, сила
- **Чередование** → Динамика, разнообразие, визуальный ритм

## 💡 Рекомендации по использованию

### Используйте тёмную тему если:
- ✅ Аудитория просматривает сайт вечером/ночью
- ✅ Нужна атмосфера серьёзности и доверия
- ✅ Важен премиальный, элегантный образ
- ✅ Контент требует максимальной концентрации

### Используйте светлую тему если:
- ✅ Аудитория активна днём
- ✅ Нужна энергичная, позитивная атмосфера
- ✅ Важен призыв к действию и активности
- ✅ Хотите создать запоминающееся впечатление

## 🔧 Техническая реализация

### Общие элементы (неизменные):
- 🎬 **Hero видео**: Одинаковое для обеих тем
- 📝 **Типографика**: Montserrat + Inter
- 🎨 **Базовая палитра**: Одна и та же (из видео)
- 🖼️ **Layout**: Идентичная структура
- ⚡ **Анимации**: Те же эффекты hover/transition

### Различия (через CSS переменные):
- `--cream-bg`: Цвет фона body
- `--card-bg`: Цвет карточек
- `--text-dark` / `--text-light`: Цвет текста
- `.project-card:nth-child(even)`: Альтернативные карточки

**Преимущество**: Переключение темы = изменение нескольких CSS переменных

## 📋 Checklist при смене темы

- [ ] Обновить `:root` переменные в `styles.css`
- [ ] Изменить `body { color: ... }`
- [ ] Обновить `.site-header { background-color: ... }`
- [ ] Добавить/убрать `.project-card:nth-child(even)`
- [ ] Пересобрать: `npm run build`
- [ ] Перезапустить: `pm2 restart human-time`
- [ ] Проверить в браузере с Ctrl+F5
- [ ] Обновить комментарий в `:root` (Вариант 1/2)
- [ ] Закоммитить изменения в git
